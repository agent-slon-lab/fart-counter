# 💨 Счётчик Пуков (Fart Counter)

> **Бесплатное офлайн PWA для отслеживания пуков.** Юмор, статистика, 23 достижения, дневник питания, погода, мировая карта. Без рекламы, без слежки.

**Версия: 1.2.2**

🌐 **English version:** [README.en.md](./README.en.md)

---

## 📱 Установить на телефон

Откройте приложение на телефоне и добавьте на главный экран:

**🌐 Ссылка:** [fart-counter-lake.vercel.app](https://fart-counter-lake.vercel.app/)

- 🤖 **Android (Chrome):** открыть ссылку → меню (⋮) → «Установить приложение»
- 🍎 **iPhone (Safari):** открыть ссылку → Поделиться → «На экран Домой»

---

## 🌟 Возможности

- 📊 **Счётчик пуков** с большой кнопкой +1 и анимацией
- 🏷️ **11 тегов**: Тихий, Вонючий, Громкий, Долгий, В туалете, Случайно, Шептун, Прострел, Музыкальный, Волна, Лягушачий
- 🔊 **14 звуков** + авто-выбор по тегу
- 🏆 **23 достижения** (20 + 3 легендарных)
- 📈 **Статистика** с графиками (Неделя/Месяц/Год) + экспорт CSV/JSON
- 📅 **Журнал** с календарём
- 🍔 **Дневник питания** — найди свои триггеры
- 😊 **Дневник настроения**
- 🌤️ **Погода и пуки** — смешные корреляции
- 🔮 **Пук-предсказатель**
- 🌍 **Мировой рейтинг** (анонимно)
- 💡 **100 фактов** на 7 языках
- 📤 **Share-карточки** (Spotify Wrapped стиль)
- 🎨 **Темы** + 4 акцента
- 🌐 **7 языков**: 🇷🇺 🇬🇧 🇪🇸 🇵🇹 🇩🇪 🇫🇷 🇮🇳
- 📱 **PWA**: офлайн, установка на телефон, уведомления
- 🔄 **Автообновление** приложения

## 🛠 Технологии

| | |
|---|---|
| Framework | Next.js 16 (App Router) |
| Язык | TypeScript 5 |
| Стили | Tailwind CSS 4 + shadcn/ui |
| State | Zustand + localStorage |
| Графики | Recharts |
| Анимации | Framer Motion |
| Звуки | Web Audio API (синтез) |
| QR-код | собственная реализация (0 зависимостей) |

## 📁 Структура

```
src/
├── app/
│   ├── layout.tsx          # Корневой layout
│   ├── page.tsx            # Приложение (PWA)
│   └── landing/            # Лендинг для SEO
├── components/
│   ├── app/                # Экраны
│   ├── pwa/                # Install prompt, Update banner
│   └── ui/                 # shadcn/ui
├── lib/
│   ├── i18n.ts             # 7 языков
│   ├── store.ts            # Zustand store
│   ├── achievements.ts     # 23 достижения
│   ├── sounds.ts           # 14 звуков
│   ├── facts.ts            # 100 фактов
│   ├── version.ts          # Проверка обновлений
│   ├── notifications.ts    # Уведомления
│   ├── qr.ts               # QR-генератор
│   └── export.ts           # CSV/JSON экспорт
└── hooks/
public/
├── manifest.json           # PWA манифест
├── sw.js                   # Service Worker
├── version.json            # Текущая версия
└── icon-*.png              # Иконки
```
