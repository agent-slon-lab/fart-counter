### Исправлено

## [1.2.0] — 2026-07-16

### 🎉 Живые приветствия + исправления

### Добавлено
- 🌟 **Живые приветствия** — полноэкранный попап с мордочкой при открытии приложения:
  - Утренние (5:00-11:00), дневные (11:00-17:00), вечерние (17:00-21:00), ночные (21:00-5:00)

  - Специальные сообщения: 0 пуков, мало пуков, норма (10-20), перебор (>20), нет воды
  - Показывается раз в день (сохраняется в localStorage)
- 📡 **Sonner Toaster** — добавлен в layout (теперь toast-уведомления работают!)
- 🌐 **IP-геолокация fallback** — если GPS недоступен/отключён, погода определяется по IP

### Исправлено
- 🔔 **Toast "Ссылка скопирована"** — теперь работает (добавлен Sonner Toaster)
- 🌤️ **Погода** — добавлен fallback на IP-геолокацию (ipapi.co) если GPS не работает
- 📊 **Недельный цикл** — исправлена ошибка с делением на 0, улучшена отрисовка столбцов
- 📋 **Копирование ссылки** — добавлен fallback через execCommand для старых браузеров

## Как обновлять changelog

При новом релизе добавьте секцию вверху:

```
## [1.1.0] — ГГГГ-ММ-ДД

### Добавлено
- Новая фича

### Изменено
- Что-то улучшено

### Исправлено
- Баг такой-то
```

И обновите версию в:
1. `package.json` → `"version": "1.1.0"`
2. `public/manifest.json` → `"version": "1.1.0"`
3. `public/version.json` → `"version": "1.1.0"`
4. `public/sw.js` → `const CACHE = "fart-counter-v1.1.0"`
5. `src/lib/version.ts` → `export const APP_VERSION = "1.1.0"`
6. В `src/lib/i18n.ts` → `about_text: "Счётчик Пуков v1.1.0 — ..."`
